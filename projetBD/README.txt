Listes des fichiers :

* create_table.sql : script  qui contient toutes les créations des tables et leurs contraintes

* insert_table.sql : script qui contient toutes les insertions des tables

* create_trigger.sql : script qui contient toutes les fonctions et triggers

* test.sql : script qui permet de faire quelques tests sur des scénarios déclenchés par les triggers(par défaut tout est entouré de commentaires)

* test_emprunt.sql : script qui permet de simuler l'emprunt d'un client par entrée au clavier

* test_reservation.sql : script qui permet de simuler la réservation d'un client par entrée au clavier

* test_inscription.sql : script qui permet de simuler l'inscription d'un nouveau client par entrée au clavier

* test_chercheExemplaire.sql : script qui permet de simuler la recherche d'un exemplaire disponible dans une bibliothèque par entrée au clavier

* test_rendreExemplaire.sql : script qui permet de simuler le rendu d'un exemplaire par un client par entrée au clavier

* test_remboursement.sql : script qui permet de simuler le remboursement d'un client par entrée au clavier

* test_statistiquesEmprunt.sql : script qui permet de connaitre le pourcentage d'emprunt d'une bibliothèque choisi par entrée au clavier

* test_renouvelExemplaire.sql : script qui permet de renouveler l'emprunt d'un exemplaire par entrée au clavier

* tets_renouvelInscription.sql : script qui permet de renouveler l'abonnement d'un client par entrée au clavier
